# FNF: A Memster Mod (Vs Memster DEMO)

Simple Friday Night Funkin' oneshot mod based on two OCs: **Memster** and **Ink**.

## What it has

- A short storyline introducing Memster and Ink.
- Memster's goofy and comedic vibe for a fun, light-hearted experience.
- Made with love for players who want a funni mod.

## Engine

This mod is for **Psych Engine 0.7 only**.
